========================================
OVERVIEW
========

mini-grades is a simple command-line grade manager.

It allows users to store and manage student grades.
Users can initialize the system, add grades, list grades,
update grades, and delete grades.

All data is stored inside a directory called .minigrades
in the current working directory.

========================================
COMMANDS
========

--- init ---
Usage: python minigrades.py init

Behavior:
Creates a directory called .minigrades and an empty file named grades.dat.

Output:
Initialized empty grade system in .minigrades/

If the directory already exists:
Already initialized

--- add ---
Usage: python minigrades.py add "Student Name" 85

Behavior:
Adds a new grade record for the student.

Each record contains:
id | student_name | grade | date

Note:
Date is currently not stored in the system (planned for future version).

Output example:
Added grade #1 for Alice

--- list ---
Usage: python minigrades.py list

Behavior:
Displays all grade records.

Output format example:

[1] Alice - 85
[2] Bob - 90

If there are no grades in the system:
No grades found.

--- update ---
Usage: python minigrades.py update 1 95

Behavior:
Updates the grade for the given record ID.

Output:
Updated grade #1 to 95

If the ID does not exist:
Grade #1 not found.

--- delete ---
Usage: python minigrades.py delete 1

Behavior:
Deletes the grade record with the given ID.

Output:
Deleted grade #1

If the ID does not exist:
Grade #1 not found.

========================================
DATA FORMAT
===========

File:
.minigrades/grades.dat

Format:
One record per line separated by |

Example:

1|Alice|85|2026-03-16
2|Bob|90|2026-03-16

Fields:
id | student_name | grade | date

Note:
The current implementation does not fully use the "date" field yet.
It will be added in future versions.

========================================
ERROR HANDLING
==============

If a command is used before initialization:

Not initialized. Run: python minigrades.py init

If an unknown command is entered:

Unknown command: <command>

If arguments are missing:

Usage: python minigrades.py <command> [args]

Note:
Error handling behavior has been clarified for missing arguments.
